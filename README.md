# Construction Site Data Repository

This repository contains synthetic data for a construction project tracking system. The data includes project overview, tasks, team responsibilities, and progress reports.

## Files

- `project-overview.md` - Overall project information, milestones, budget
- `tasks.json` - Current tasks with status, assignments, and progress
- `team-responsibilities.md` - Team structure and responsibilities
- `progress-report.md` - Weekly progress updates

## Data Structure

### Tasks
Each task includes:
- ID, title, description
- Status (pending, in_progress, completed)
- Priority (low, medium, high, critical)
- Assigned team member
- Due date
- Progress percentage
- Location
- Estimated vs actual hours

### Team Members
Key roles:
- Project Manager
- Structural Lead
- MEP Coordinator
- Electrical Lead
- Plumbing Lead
- Exterior Lead
- Parking Lead
- Elevator Specialist
- Safety Officer
- QC Manager

## Usage

This data can be used to:
- Track construction progress
- Manage task assignments
- Monitor budget and schedule
- Generate reports
- Query project status

## Updates

Data is updated based on:
- Daily progress reports
- Task completions
- Team assignments
- Schedule changes
- Budget updates
